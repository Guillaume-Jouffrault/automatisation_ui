automatisation_ui

Ce projet consiste en l'installation automatique d'un installer windows.

Ce projet a 2 objectifs : 
1) Installer le programme en appuyant sur le moins de boutons possible 
2) Installer le programme en appuyant sur le plus de boutons possible 

Le premier objectif permet d'installer des malwares sur une VM pour les étudier.
Le second ojectif permet de déclancher les programmes malveillants camouflés dans l'installer.

Ce projet est réalisé en partenariat avec Cyber-Detect.
